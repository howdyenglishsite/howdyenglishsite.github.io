اضغط لمشاهدة الفيديو

1️⃣ التحيات اليومية العامة

Hello → مرحباً
Hi → أهلاً
Hey → هاي (غير رسمي جداً)
Good morning → صباح الخير
Good afternoon → مساء الخير (من الظهر حتى قبل المغرب)
Good evening → مساء الخير (بعد المغرب)
Good night → تصبح على خير (تستخدم عند وداع شخص قبل النوم)
How are you? → كيف حالك؟
How’s it going? → كيف الأمور؟ (غير رسمي)
How do you do? → كيف حالك؟ (رسمي جداً)
What’s up? / Wassup? → ما الأخبار؟ / شو في؟

2️⃣ تحيات غير رسمية بين الأصدقاء

Yo → هاي / أهلاً (غير رسمي جداً، بين شباب)
Sup → اختصار لـ “What’s up?”
Hiya → أهلاً (غير رسمي)
Hey there → مرحباً هناك
Long time no see → لم نلتق منذ فترة
Howdy → مرحباً (شائع في الولايات المتحدة، له طابع ودود)
How’s everything? → كيف كل شيء؟

3️⃣ تحيات رسمية أو مهذبة

Greetings → تحية رسمية
Good to see you → سررت برؤيتك
Pleased to meet you → سعيد بلقائك (أول لقاء رسمي)
How have you been? → كيف كنت منذ آخر لقاء؟
It’s a pleasure to meet you → تشرفت بلقائك

4️⃣ تحيات صباحية/مسائية محددة

Morning! → صباح الخير (غير رسمي)
Afternoon! → مساء الخير (غير رسمي)
Evening! → مساء الخير (غير رسمي)
Rise and shine! → استيقظ وتألّق! (مرح صباحي)
Have a nice day! → أتمنى لك يوماً سعيداً

5️⃣ تحيات وداع

Bye / Goodbye → وداعاً
See you → أراك لاحقاً
See you later → أراك لاحقاً
See you soon → أراك قريباً
Take care → اعتنِ بنفسك
Catch you later → أراك لاحقاً (غير رسمي)
Farewell → وداعاً (رسمي وأدبي)
Have a good one → أتمنى لك وقتاً طيباً (غير رسمي)

6️⃣ تحيات للمكالمات أو الرسائل

Hello, this is [Name] speaking → مرحباً، معك [الاسم] (للمكالمات)
Hi, how are you doing? → أهلاً، كيف حالك؟
Hey! Long time no chat → هاي! لم نتحدث منذ فترة
Greetings of the day → تحية اليوم (رسمي، للبريد)
