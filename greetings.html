اضغط لمشاهدة الفيديو

1️⃣ التحيات اليومية العامة

Hello → مرحباً
Hi → أهلاً
Hey → هاي (غير رسمي جداً)
Good morning → صباح الخير
Good afternoon → مساء الخير
Good evening → مساء الخير
Good night → تصبح على خير
How are you? → كيف حالك؟
How’s it going? → كيف الأمور؟
How do you do? → كيف حالك؟ (رسمي جداً)
What’s up? → ما الأخبار؟

2️⃣ تحيات غير رسمية بين الأصدقاء

Yo → هاي / أهلاً
Sup → What’s up
Hiya → أهلاً
Hey there → مرحباً هناك
Long time no see → لم نلتق منذ فترة
Howdy → مرحباً
How’s everything? → كيف كل شيء؟

3️⃣ تحيات رسمية أو مهذبة

Greetings → تحية رسمية
Good to see you → سررت برؤيتك
Pleased to meet you → سعيد بلقائك
How have you been? → كيف كنت منذ آخر لقاء؟
It’s a pleasure to meet you → تشرفت بلقائك

4️⃣ تحيات صباحية/مسائية محددة

Morning! → صباح الخير
Afternoon! → مساء الخير
Evening! → مساء الخير
Rise and shine! → استيقظ وتألّق!
Have a nice day! → أتمنى لك يوماً سعيداً

5️⃣ تحيات وداع

Bye / Goodbye → وداعاً
See you → أراك لاحقاً
See you later → أراك لاحقاً
See you soon → أراك قريباً
Take care → اعتنِ بنفسك
Catch you later → أراك لاحقاً
Farewell → وداعاً
Have a good one → أتمنى لك وقتاً طيباً

6️⃣ تحيات للمكالمات أو الرسائل

Hello, this is [Name] speaking → مرحباً، معك [الاسم]
Hi, how are you doing? → أهلاً، كيف حالك؟
Hey! Long time no chat → هاي! لم نتحدث منذ فترة
Greetings of the day → تحية اليوم
